# Road Safety Dar — User Guide

## 1. Public Dashboard
Visit `/dashboard/` to see the city-wide map, heatmap, and charts.

## 2. Submit a Report
Tap **Report Accident** → fill the form → **Use my current location** → Submit.
Reports are visible immediately on the public map.

## 3. Authority Dashboard
Visit `/authority/` for the high-risk hour profile and recommendations.

## 4. Admin
`/admin/` — verify reports, manage junctions, export CSV.
