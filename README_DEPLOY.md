# 🚀 Deployment Reference — Road Safety Dar es Salaam

## Live URL
https://roadsafety-dar.onrender.com

## Architecture
```
Browser → Render.com (gunicorn + WhiteNoise) → Supabase PostgreSQL + PostGIS
```

## Render.com Dashboard
https://dashboard.render.com → roadsafety-dar service

## Environment Variables (set in Render dashboard)

| Variable | Where to get it |
|---|---|
| DJANGO_SECRET_KEY | Auto-generated by Render |
| DJANGO_ENV | Set to: prod |
| DEBUG | Set to: False |
| ALLOWED_HOSTS | roadsafety-dar.onrender.com |
| DATABASE_URL | Supabase → Settings → Database → URI |
| SUPABASE_URL | Supabase → Settings → API → Project URL |
| SUPABASE_ANON_KEY | Supabase → Settings → API → anon public |
| SUPABASE_SERVICE_KEY | Supabase → Settings → API → service_role |
| SUPABASE_JWT_SECRET | Supabase → Settings → API → JWT Secret |

## Deploy a new version
```bash
git add .
git commit -m "your message"
git push origin main
# Render auto-deploys from GitHub main branch
```

## Manual redeploy
Render dashboard → roadsafety-dar → Manual Deploy → Deploy latest commit

## Check logs
Render dashboard → roadsafety-dar → Logs

## Run smoke tests
```bash
python scripts/health_check.py https://roadsafety-dar.onrender.com
```

## Rollback
Render dashboard → roadsafety-dar → Events → click any previous deploy → Rollback
